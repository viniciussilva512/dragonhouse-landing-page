#  DragonHouse

Landing page desenvolvida para um desafio de HTML e CSS.

A DragonHouse é uma organização fictícia inspirada em universos de fantasia medieval, dedicada à adoção de dragões lendários. O projeto foi criado com foco em criatividade, design visual e organização de conteúdo utilizando apenas HTML e CSS.

##  Funcionalidades

* Landing page responsiva
* Design inspirado em fantasia medieval
* Cards de dragões lendários
* Ranking de adoções
* Crônicas de adoção em formato de pergaminho
* Ícones com Font Awesome
* Animações e efeitos visuais em CSS

##  Tecnologias Utilizadas

* HTML5
* CSS3
* Font Awesome
* Google Fonts

##  Objetivo

Desenvolver uma landing page criativa utilizando apenas HTML e CSS, explorando conceitos de layout, estilização e responsividade.
